How to get Short User Token

1

Go to the Graph API Explorer .

2

In the right panel, select your Meta App from the dropdown.

3

Under "User or Page", select Get User Token.

Make sure you grant the necessary permissions (like `pages_manage_posts`, `pages_read_engagement` and `pages_show_list`).

4

Copy the generated token from the Access Token box and paste it here.
